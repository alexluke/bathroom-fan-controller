# Bathroom Fan Controller

## Required Hardware
* Adafruit Trinket 5v
* DHT11 or DHT22

### Optional Hardware

Used for debugging

* Adafruit LCD Backpack
* LCD Screen

## Require Libraries
* [TinyDHT](https://github.com/adafruit/TinyDHT)

### Optional Libraries

Used for debugging

* [TinyLiquidCrystal](https://github.com/adafruit/TinyLiquidCrystal)
* [TinyWireM](https://github.com/adafruit/TinyWireM)
